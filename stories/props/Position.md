List of positon props, and example.

## 	Definition

CSS properties that we use for background HTML element.
Property that adds to the component are:

- `$position`
- `$top`
- `$right`
- `$bottom`
- `$left`
- `$inset`
- `$zIndex`
- `$transform`
- `$alignSelf`
- `$order`


## Usage 

```jsx
<Wrapper
	$height="200px"
	$width="200px"
	$backgroundColor="yellow"
	$position="relative"
>
	<Wrapper
		$height="100px"
		$width="100px"
		$backgroundColor="backgroundLight"
		$position="absolute"
		$top="25%"
		$left="75%"
	/>
</Wrapper>
```
